Day and Night classification of images:
We use 3 methods to classify images given in the folders day and night:
1)Threshholding of intensity values in the histogram
2)Taking mean of rgb values together and setting a combined threshhold
3)Taking mean of r,g,b values separately and setting separate threshholds.

How to run code?
1)Click on DayOrNight .m file
2)Run it.
